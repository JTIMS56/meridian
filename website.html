05 // NLP Pipeline
Ten Analytical Methods
Combines classical statistical NLP with transformer-based semantic analysis and entity intelligence. Full-text article extraction provides 641-word average document depth.
Classical
TF-IDF
Term frequency-inverse document frequency with ngrams(1,3). Extracts top 30 corpus signal terms weighted by statistical importance.
scikit-learn // TfidfVectorizer
Classical
LDA Topic Modeling
Latent Dirichlet Allocation discovers 6 latent topics across the combined GDELT + social corpus using word co-occurrence patterns.
scikit-learn // LatentDirichletAllocation
Classical
VADER Sentiment
Lexicon-based sentiment analysis on translated text. Produces positive/neutral/negative distribution and by-language breakdowns.
vaderSentiment // SentimentIntensityAnalyzer
Classical
spaCy NER
Named entity recognition extracting PERSON, ORG, GPE, and LOC entities. Surfaces key actors and locations mentioned in the corpus.
spaCy // en_core_web_sm
Classical
ARIMA Forecast
Time-series volume forecasting with ARIMA(2,1,1) on 30-day timeline data. Projects article volume at 7, 30, 90, and 365-day horizons.
statsmodels // ARIMA
Semantic
BERTopic
Transformer-based topic modeling using sentence embeddings, UMAP dimensionality reduction, and HDBSCAN clustering. Understands meaning, not just word frequency.
sentence-transformers // all-MiniLM-L6-v2
Semantic
Zero-Shot IO Framing
NLI-based classification into 8 IO narrative frames without domain-specific training: influence operation, sovereignty threat, debt trap warning, and more.
facebook // bart-large-mnli
Semantic
Narrative Echo Detection
Computes cosine similarity between corpus documents and 5 reference IO narratives. Detects which narratives dominate the information environment.
sentence-transformers // cosine similarity
Semantic
RoBERTa Sentiment
Transformer sentiment fine-tuned for social text. Methodological triangulation against VADER reveals factual tone vs adversarial vocabulary divergence.
cardiffnlp // twitter-roberta-base-sentiment
Entity Intel
Entity Intelligence
Watchlist-based tracking of named persons, organizations, and locations per profile with mention counts, sentiment context, and co-occurrence mapping.
spaCy + VADER // per-profile watchlists
